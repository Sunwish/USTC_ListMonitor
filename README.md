# 中科大官网硕士招生公告监视提示助手

## 使用说明

- 首次使用需配置提示 SCKEY，前往 http://sc.ftqq.com/ 生成你的唯一 SCKEY。
- 监视时间间隔为 15 秒，检测到新公告将通过 SCKEY 绑定的微信发送提醒，请在 Server 酱官网配置微信推送（绑定微信关注其公众号以接收提醒推送）。
- 简易 nohup 部署：`nohup dotnet USTC_ListMonitor.dll`
